# Generated Flight Reports

This folder contains sample JSON and Markdown reports generated by the UAV Telemetry Ground Station.

Reports are intentionally committed for portfolio review so a recruiter or hiring manager can quickly see example output without running the project.

Useful examples:

- `normal_report.md`
- `unsafe_armed_report.md`
- `mixed_faults_report.md`
- `demo_mixed_faults.md`
