# Resume Bullets: UAV Telemetry Ground Station

Use one or two of these depending on the resume version.

## Strong version

- Built a Python UAV telemetry ground station for simulated/replayed drone data, monitoring GPS, altitude, battery, current draw, flight mode, arming state, rangefinder distance, link quality, and warning states.
- Implemented warning detection, CSV logging, flight-log replay, and JSON/Markdown post-run reports for low battery, GPS loss, high current, low link quality, sensor faults, unsafe arming state, and telemetry timeout conditions.
- Designed the project as a field-test/debugging workflow for future MAVLink/PX4/ArduPilot integration with physical UAV hardware.

## Short version

- Developed a Python UAV telemetry and reporting tool with simulated fault scenarios, safety warnings, CSV log replay, and JSON/Markdown flight summaries for robotics field-test workflows.
